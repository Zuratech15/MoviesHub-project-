# MoviesHub

MoviesHub is a front-end web project created for educational purposes.  
The project demonstrates responsive web design, clean code structure, and dynamic content loading using a public API.

## Project Overview

The website displays a list of popular movies fetched dynamically from the TMDB API.  
It includes multiple pages, responsive layouts for different screen sizes, and interactive elements implemented with JavaScript.

## Features

- Responsive design for desktop (1440px) and mobile (420px)
- Semantic HTML5 structure
- Clean and organized file structure
- Dynamic navigation rendered with JavaScript
- Movie data fetched using the Fetch API
- Fixed header on scroll using JavaScript
- Mobile burger menu functionality
- CSS variables for consistent color management
- CSS animations using @keyframes
- Hover effects for interactive elements
- External redirection to official movie pages (TMDB)

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- TMDB API
- GitHub Pages (hosting)

## API Integration

Movie data is retrieved dynamically from the TMDB public API using the Fetch API.  
The content is not hardcoded, which allows the project to display real-time data and remain scalable.
