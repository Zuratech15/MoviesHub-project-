      Technologies, API integration, and learning outcomes

      Technologies Used

      The project is built using semantic HTML5 to ensure clear structure and accessibility. CSS is used to create responsive layouts and maintain visual consistency across different devices. JavaScript handles dynamic behavior and API communication.

      API Integration

      Movie data is fetched from the TMDB public API instead of being hardcoded. This allows the content to remain dynamic, scalable, and aligned with real-world development practices.

      Learning Outcome

      This project helped reinforce key front-end concepts such as responsive design, API data handling, and structured layout development using modern web standards.
